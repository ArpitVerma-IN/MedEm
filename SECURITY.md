# Security Policy

## Supported Versions

Currently, MedEm is a prototype application. Only the latest version pushed to the `main` branch is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| `main`    | :white_check_mark: |
| < `main`  | :x:                |

## Reporting a Vulnerability

We take the security of MedEm seriously. If you discover a security vulnerability, please do NOT file a public issue. 

Instead, please send an email to the repository owner or submit a private vulnerability report via GitHub's "Security" tab.

Please include the following details in your report:
* A description of the vulnerability and its potential impact.
* Steps to reproduce the issue.
* Any potential mitigation or fix if you are aware of one.

We will endeavor to respond to your report within 48 hours and work with you to resolve the issue as quickly as possible.  Once a fix is implemented and successfully deployed, the vulnerability may be publicly disclosed.
